### Prerequisites

    Node.js (>= 18.17)
    npm (>= 8.55.0)

### Frotend - Runtimes, Engines, and Tools

    TypeScript (>= 5.2.2)
    Vite (>= 5.0.8)

### Running Locally

Install dependencies using npm:

    npm install

Copy .env.example to .env

    cp .env.example .env

### Start the development server

    npm run dev
